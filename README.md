# Fullstack Practice
 
Exercises and projects from the [Full Stack Open](https://fullstackopen.com/) course (University of Helsinki). The folder structure follows the parts of the course.
 
## Featured project
 
### Notes app
Full stack application with user registration and login, token-based authorization (JWT), session persistence, and full CRUD operations on notes.
 
- Frontend: [`/Part2A`](./Part2A)
- Backend: [`/Part3/Notes`](./Part3/Notes)
## Tech stack
 
React · Node.js · Express · MongoDB · JWT
